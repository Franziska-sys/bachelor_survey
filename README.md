# Survey: HTML5 Frameworks vs. Game Engines

A static single-page survey (HTML/CSS/JS) for a bachelor's thesis comparing browser-based game development approaches.

Stack: Static HTML hosted via any web server (e.g. Netlify, GitHub Pages). Survey responses are submitted to a Google Spreadsheet via Google Apps Script.

Features: 20 questions across 6 sections, progress bar, client-side validation, one-submission guard via localStorage, and a JSON file fallback if the network request fails.
